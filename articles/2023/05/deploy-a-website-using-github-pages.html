Deploy a Website Using Github Pages
Github can make as Deploy website static. Follow these steps:
1. Create a GitHub repository

The first step in deploying a website using GitHub Pages is to create a new repository

2. Add content to your repository

Then, we go clicking on the "Add file" button and choosing the files you wish to add. Ensure that all files are in the root directory of the repository. There is my file is blog. But we can add file index.html for page of number one of this site

Make file index.html for main page


Make file blog.html for page
3. Configure repository settings

After adding the necessary files, you need to configure the repository settings to enable GitHub Pages. To do this, click on the "Settings" tab and scroll down to the "GitHub Pages" section. From the "Source" dropdown menu, select "Github Acions" and save it
